# TaskFlow - 智能任务管理

一个具有职业蓝调风格的智能 Todo List 应用，集成 DeepSeek AI 助手，帮助您快速规划和完成任务。

## 📋 快速开始

### 步骤 1：安装依赖

在项目目录下运行以下命令安装所需的依赖包：

```bash
npm install
```

### 步骤 2：配置 DeepSeek API Key

1. 复制环境变量模板文件：

```bash
重命名 .env.example  -> .env
```

2. 编辑 `.env` 文件，将 `your_deepseek_api_key_here` 替换为您的实际 API Key：

```
VITE_DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

3. 获取 API Key：
   - 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
   - 注册账号并登录
   - 在控制台获取 API Key（新用户有免费额度）

### 步骤 3：启动应用

```bash
npm run dev
```

应用将在浏览器中自动打开：**http://localhost:3000**

## 📖 使用教程

### 基础功能操作

#### 添加任务
- 在顶部的输入框中输入任务内容
- 点击"添加"按钮或直接按 `Enter` 键
- 任务将立即出现在列表中

#### 完成任务
- 点击任务左侧的复选框
- 已完成的任务会显示删除线，且透明度降低
- 顶部进度条会自动更新

#### 编辑任务
- 将鼠标悬停在任务上（移动端直接点击任务）
- 点击右侧的"铅笔"图标进入编辑模式
- 修改完成后按 `Enter` 保存，或按 `ESC` 取消

#### 删除任务
- 将鼠标悬停在任务上
- 点击右侧的"垃圾桶"图标
- 任务将被永久删除

#### 筛选任务
使用顶部的筛选按钮：
- **全部** - 显示所有任务
- **进行中** - 仅显示未完成的任务
- **已完成** - 仅显示已完成的任务

#### 清除已完成
当有已完成的任务时，底部会出现"清除已完成任务"按钮，点击可批量删除所有已完成的任务。

### AI 智能助手使用

#### 打开 AI 助手
点击右下角的蓝色圆形按钮（带有聊天图标），AI 助手对话框将从底部弹出。

#### 使用 AI 生成任务
1. 在对话框中输入您的目标或需求
2. 点击发送按钮或按 `Enter` 键
3. AI 会自动分析并生成多个相关的待办事项
4. 生成的任务会自动添加到您的任务列表中

#### AI 使用示例
```
"帮我制定一个健身计划"
→ 生成：每周跑步3次、做俯卧撑50个、健身房训练等任务

"如何学习 TypeScript"
→ 生成：安装TypeScript、学习基础语法、实践项目等任务

"准备去日本旅行"
→ 生成：办理护照、预订机票、查找酒店、制定行程等任务
```

#### AI 对话功能
- 您可以与 AI 进行多轮对话
- AI 会根据上下文提供更精确的建议
- 支持自然语言交互，像聊天一样轻松

## 💡 小技巧

- **自动保存**：所有数据自动保存到浏览器，关闭页面也不会丢失
- **快捷键**：
  - `Enter` - 添加任务 / 发送消息 / 保存编辑
  - `ESC` - 取消编辑
  - `Shift + Enter` - AI 助手中换行
- **庆祝动画**：完成所有任务时会有彩带庆祝效果
- **进度追踪**：顶部进度条实时显示完成百分比

## ❓ 常见问题

**Q: AI 功能不工作？**
A: 请确保已正确配置 `.env` 文件中的 `VITE_DEEPSEEK_API_KEY`。

**Q: 数据丢失了怎么办？**
A: 数据保存在浏览器的 localStorage 中，清除浏览器数据会导致丢失。建议定期备份重要任务。
